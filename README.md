# Upload program ke streamlit

1. Masuk ke halaman https://streamlit.io/
2. Pada pojok kanan atas pilih Free kemudian sign in ke streamlit community.
3. Login menggunakan email address atau metode login lainnya yang disukai.
4. Setelah masuk ke halaman main menu, cek di bagian kanan atas, akan terdapat button Create App.
5. Setelah Create App, pilih Deploy a Public App from GitHub.
6. Isi informasi yang diminta dari GitHub.
7. Pastikan informasi yang dimasukkan sudah sesuai, kemudian pilih Deploy.
8. Setelah berhasil di deploy, tambahkan link url streamlit ke bagian readme agar dapat diakses dengan mudah.
